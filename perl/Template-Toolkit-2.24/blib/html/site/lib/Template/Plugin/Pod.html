NAME

Template::Plugin::Pod - Plugin interface to Pod::POM (Pod Object Model)

SYNOPSIS

    [% USE Pod(podfile) %]
    
    [% FOREACH head1 = Pod.head1;
         FOREACH head2 = head1/head2;
           ...
         END;
       END
    %]

DESCRIPTION

This plugin is an interface to the the Pod::POM manpage module.
